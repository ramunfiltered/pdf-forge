# PDF Forge

A modern Chrome extension for editing PDFs completely offline.

## Features

- View PDFs
- Annotate PDFs
- Highlight text
- Draw on PDFs
- Merge PDFs
- Split PDFs
- Rotate PDFs
- Compress PDFs
- Watermark PDFs
- Convert PDFs
- Offline Processing

## Installation

1. Clone this repository.
2. Open Chrome.
3. Navigate to `chrome://extensions`.
4. Enable **Developer mode**.
5. Click **Load unpacked**.
6. Select this folder.

## Tech Stack

- HTML
- CSS
- JavaScript
- PDF.js
- jsPDF
- Chrome Extension Manifest V3
